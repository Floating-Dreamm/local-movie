# 本地视频播放器

一个使用前端技术和Python Flask搭建的本地网页视频播放器。

## 功能特点

- 📺 支持多种视频格式（MP4, AVI, MKV, MOV, WMV, FLV, WebM）
- 📋 支持选择本地文件夹，自动扫描并列出可播放的视频文件
- 📁 支持子文件夹分类展示，清晰组织视频内容
- 🖼️ 自动生成视频缩略图，直观预览视频内容
- 🎬 点击视频列表即可播放
- 💪 大文件优化：
  - 流式加载，避免一次性加载整个视频文件
  - 智能缩略图生成，大文件使用默认缩略图
  - 延迟加载机制，优先处理小文件
  - 超时保护，防止加载卡住
- 📊 显示视频文件大小信息
- 🎨 现代化的UI设计
- 📱 响应式布局，支持移动端

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **后端**: Python Flask

## 项目结构

```
local-movie/
├── app.py                 # Python Flask后端代码
├── requirements.txt       # 项目依赖
├── templates/
│   └── index.html         # 前端HTML页面
├── static/
│   ├── css/
│   │   └── style.css      # CSS样式文件
│   └── js/
│       └── script.js      # JavaScript脚本
└── videos/                # 视频文件目录
```

## 安装和使用

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 添加视频文件

将您的视频文件（支持MP4, AVI, MKV, MOV格式）放入`videos`目录中。

### 3. 启动服务器

```bash
python app.py
```

### 4. 访问播放器

在浏览器中访问 `http://localhost:5000`

## 使用说明

1. 启动服务器后，浏览器会自动加载videos目录下的所有视频文件
2. 在视频列表中点击您想要观看的视频
3. 使用视频播放器的控制栏进行播放、暂停、音量调节等操作

## 注意事项

- 确保您的视频文件格式受浏览器支持
- 较大的视频文件可能需要更长的加载时间
- 建议使用MP4格式以获得最佳兼容性
